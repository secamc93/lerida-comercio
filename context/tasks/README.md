# tasks/

Tareas en curso, pendientes y decisiones técnicas.

## Convención de nombres

`YYYY-MM-DD-titulo-corto.md` — por ejemplo `2026-05-15-migrar-jwt-a-cookies.md`.

## Plantilla sugerida

```markdown
# Título

**Fecha:** 2026-MM-DD
**Estado:** pendiente | en curso | bloqueada | completada

## Contexto
Por qué estamos haciendo esto.

## Plan
1. Paso 1
2. Paso 2

## Notas
Lo que aprendí mientras avanzaba, decisiones tomadas, links útiles.

## Pendiente
- [ ] cosa 1
- [ ] cosa 2
```

Cuando se complete, mover a `done/` o eliminar.
